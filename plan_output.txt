6. OpenAI GPT-4.1-mini Integration Pattern
Since there is no backend, all OpenAI calls go directly from the React frontend. Store the API key in .env.local as VITE_OPENAI_API_KEY. Never commit this file.
// src/lib/openai.ts
const callGPT = async (systemPrompt: string, userMessage: string) => {
  const res = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${import.meta.env.VITE_OPENAI_API_KEY}`
    },
    body: JSON.stringify({
      model: 'gpt-4.1-mini',
      messages: [
        { role: 'system', content: systemPrompt },
        { role: 'user', content: userMessage }
      ],
      max_tokens: 500
    })
  });
  const data = await res.json();
  return data.choices[0].message.content;
};
System Prompts per Module
Module
GPT-4.1-mini System Prompt
Grievance Triage
You are a grievance classification AI. Given a student complaint, return JSON: { category: string, urgency: 1-5, sentiment: 'positive'|'neutral'|'negative', department: string, summary: string }. Only return JSON, no other text.
Maintenance
You are a maintenance severity AI. Given an issue description, return JSON: { severity: 1-5, category: 'electrical'|'plumbing'|'furniture'|'civil', priority: 'critical'|'high'|'medium'|'low', estimatedTime: string }. Only return JSON.
Policy Navigator
You are a campus policy assistant. Answer ONLY using the document text provided. End every answer with: SOURCE: [document name, approximate section]. If the answer is not in the document, say 'This information is not in the provided policy documents.'
Lost & Found
You are a lost and found matching AI. Given two item descriptions, return JSON: { matchScore: 0-100, matchReason: string, confidence: 'high'|'medium'|'low' }. Only return JSON.
